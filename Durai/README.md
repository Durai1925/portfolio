# Personal Portfolio

A simple static portfolio website scaffold. Update `index.html` with your resume details, replace placeholder text, and add your own projects.

## Getting started

1. Open `index.html` in your browser.
2. Edit the sections in `index.html` to match your name, experience, education, skills, and projects.
3. Customize `styles.css` if you want a different look.

## Sections to update

- About Me
- Experience
- Education
- Skills
- Projects
- Contact
